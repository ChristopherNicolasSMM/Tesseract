# 04 — Perguntas Frequentes

**P: Esqueci minha senha, e agora?**
R: Ainda não existe um fluxo de "esqueci minha senha" — peça para o
administrador redefinir.

**P: Por que só vejo o Banco de Levedura?**
R: O sistema está sendo construído por etapas. As outras áreas
(receitas, estoque, dispositivos) chegam nas próximas atualizações.

**P: Apaguei uma cepa por engano, perdi os dados?**
R: Se você usou "mover para lixeira", não — é só restaurar. Se foi
"excluir permanentemente", infelizmente os dados são removidos de
verdade e não há como recuperar.
