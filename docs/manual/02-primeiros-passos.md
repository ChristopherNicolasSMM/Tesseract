# 02 — Primeiros Passos

## 1. Acessando o sistema

Por enquanto, o acesso é feito via linha de comando (ainda não há uma
tela de login pronta visualmente). Peça para quem instalou o sistema
criar seu usuário.

## 2. Login

Use seu usuário e senha na tela de login (ou, enquanto a tela visual
não estiver pronta, via uma chamada de API que o time técnico pode
fazer por você).

## 3. Primeira tela

Depois de logado, a primeira área disponível é o **Banco de
Levedura** — lista de cepas cadastradas.
